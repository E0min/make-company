PRD: 간단한 메모 기능
1. 개요
사용자가 짧은 텍스트 메모를 작성·조회·수정·삭제할 수 있는 기본 CRUD 기능.
2. 목표
- 아이디어/할 일을 빠르게 기록
- 최소 클릭으로 메모 관리
3. 유저 스토리
- 사용자로서 새 메모를 작성하고 저장할 수 있다.
- 사용자로서 저장된 메모 목록을 최신순으로 조회할 수 있다.
- 사용자로서 기존 메모를 수정할 수 있다.
- 사용자로서 불필요한 메모를 삭제할 수 있다.
4. 기능 요구사항
┌─────┬───────────┬────────────────────────────────────┐
5. 데이터 모델
Memo { id, title?, content, createdAt, updatedAt }
6. API (예시)
- POST /memos
- GET /memos?page=
- GET /memos/:id
- PUT /memos/:id
- DELETE /memos/:id
7. 비범위 (Out of Scope)
- 사용자 인증/공유, 태그, 첨부파일, 검색 (v2 이후)
8. 성공 지표
- 메모 작성 성공률 ≥ 99%
- 목록 응답 시간 < 300ms
@backend @frontend 위 스펙 기반으로 구현 부탁드립니다.
